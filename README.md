## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Amankohare0514/Portfolio-2024

2. Navigate to the project directory:

   ```bash
   cd Portfolio-2024

3. Install dependencies:

   ```bash
   npm i

4. Start the server

   ```bash
   npm start

Open your browser and visit http://localhost:3000 .
